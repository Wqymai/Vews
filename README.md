#一、功能
* 根据新闻频道获取新闻列表
* 点击某一条新闻跳到新闻详情页面
* 新闻列表界面下拉刷新，列表底部上滑加载更多

#二、涉及技术
* (OkHttp+百度apistore)调用在线新闻API
* (Gson)gson数据解析
* (Glide)图片加载
* (SwipeRefreshLayout+RecyclerView+CardView)列表界面Item动态布局、下拉刷新，上滑加载更多
